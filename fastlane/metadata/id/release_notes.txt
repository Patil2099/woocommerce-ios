WooCommerce sudah tidak sabar untuk mengenalkan pembaruan teranyar untuk aplikasi seluler.

Selalu Siap: Misalkan, pengguna menerapkan parameter penyortiran dan/atau penyaringan pada daftar produk toko Anda lalu beralih membuka aplikasi lain. Ketika kembali, pengguna tidak ingin mengulangi proses itu lagi dari awal. Tidak masalah! Sekarang, jika pengguna membuka kembali aplikasi, mereka dapat langsung menggunakan pengaturan sebelumnya.

Penggabungan: Layar pemenuhan pesanan khusus sudah basi. Kini informasi pemenuhan pesanan dapat langsung diakses pada layar detail pesanan, begitu juga informasi mengenai biaya.

Enyahlah Bug: Kami telah memperbaiki beberapa gangguan, contohnya:

- Sekarang lembar tindakan informasi penagihan ditampilkan tanpa gangguan di iPads.
- Baris pada daftar pencarian produk tidak lagi menggunakan pemisah ganda.
- Spinner pemuatan pada tombol “lanjutkan” pada mode gelap sekarang sudah ditampilkan.
